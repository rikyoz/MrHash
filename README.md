# Mr. Hash
Mr. Hash calculates in real-time many hash strings of a given text.

## Supported Algorithms
+ MD4
+ MD5
+ SHA1
+ SHA-224, SHA-256, SHA-384, SHA-512
+ SHA3-224, SHA3-256, SHA3-384, SHA3-512
+ Tiger
+ HAVAL
+ RIPEMD
+ CRC16
+ CRC32
+ CRC64
